# 模型能力聲明（Capability Statement）

> 誠實陳述本引擎「能預測什麼、不能預測什麼」，並逐量給出信心等級與依據。
> 用於對外報告/論文的 Limitations，以及避免把工具用在它不可信的地方。

---

## 能可靠回答的問題

- 「在 X 溫區，哪種材料/成分最值得做？」（相對排序 + 最低成本）
- 「瓶頸在哪？」（顯熱、磁滯、κ、元素空間——已被分析與 GA 行為佐證）
- 「複合材料值不值得？加多少基底？」（×7–12、φ*≈0.3，定性穩健）
- 「絕對效能的『天花板』與『量級』大概多少？」（帶 ±12% 誤差條 + D12 折減）

## 不應拿來做的事

- 不可當**絕對功率密度**的精確預測（是理想化上界，~10× 高於現有原型）。
- 不可外推到**未含元素/未校準體系**而當絕對值（synthetic 外推 R²<0）。
- 不可忽略**製程/微結構**（脆性、相形成、connectivity）——這些需實測。

## 把不確定性說清楚（統計）

- Tc：真實資料 MAE±91°C（一個標準誤量級）。
- ΔS_M/ΔM：文獻 ±12%，已用 Monte Carlo 傳過整機 → P/V、η 帶誤差條
  （見 `CALIBRATED_PREDICTION.md`）。
- 絕對 P/V：系統性高估 ~10×（D12），方向已知，故給「÷10 現實估計」。

> 一句話：**本引擎是可信的「相對設計與篩選工具」，其絕對輸出帶明確、可辯護的
> 誤差條與現實折減；唯有實體量測能把絕對值最終收斂**（見 `MEASUREMENT_PROTOCOL.md`）。
