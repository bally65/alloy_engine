# GA 整機複合材料搜尋：把複合有效物性接回基因演算法

> 對應程式：`device_score.py`（複合最佳化向量化）、`gpu_ga.py`（`device_matrix`）、
> `run_search.py`（`--device-matrix`）。這是「材料 → 整機」研究的最終整合。

---

## 做了什麼

`device_power_efficiency_score` 新增 `matrix` 參數：給定高導熱基底（Cu/Al/
α-Fe）時，對每個候選 **向量化掃描最佳基底體積分率 φ\***，回傳「最佳複合材料」
的整機分數。GA 因此能對「相組成 + 自動最佳化的基底分率」整體最佳化。

用法（預設關閉，向後相容）：
```bash
# 裸相整機目標
python scripts/run_search.py --scenario 低溫廢熱_150C --mode thermomagnetic --w-device 1.0
# 複合整機目標（GA 為每候選自動找最佳 Cu 基底分率）
python scripts/run_search.py --scenario 低溫廢熱_150C --mode thermomagnetic \
    --w-device 1.0 --device-matrix Cu
```
輸出 CSV 多一欄 `matrix_frac`（各候選的最佳基底分率）。

---

## 結果（低溫情境，2 萬族群 × 40 代，快速代理）

| 整機目標 | κ 平均 | ΔM 平均 | matrix_frac 平均 | device_P (kW/m³) |
|---|---|---|---|---|
| 裸相 | 99 | 0.439 | – | 1,314 |
| 複合 (Cu) | 98 | 0.440 | **0.00** | 1,354 |

**GA 對所有候選都選了 matrix_frac = 0** —— 這是正確且有意義的物理結果：

1. **GA 的無稀土元素空間（Fe-Ni-Co…）最佳相 κ 已達 ~100 W/mK，不是 κ 受限**。
   既然頻率已高，再加 Cu 只會稀釋 ΔM，得不償失，故最佳分率為 0。
2. 這與 `device_score` 採用**含磁滯懲罰的品質頻率**一致：高頻時 Steinmetz 損耗
   讓 κ 的邊際效益封頂，所以對已高 κ 的 Fe 相，複合無益。
3. 驗證複合路徑**非恆為零**：對人造低 κ 相（85% Mn，κ≈7.8）測試，GA 路徑挑出
   φ=0.26（見整合測試），證明邏輯正確、只對真正 κ 受限的相才加基底。

---

## 核心結論：瓶頸已轉移到「元素空間」

複合材料只對 **低 κ 的稀土/一階相**（Gd₅Si₂Ge₂、Mn-Fe-P，κ=3–10）有大幅
增益（前一 PR 量化為 ×5–×22）。但這些相**含稀土或非引擎元素，GA 結構上搜不到**；
而 GA 能搜到的 Fe 系相 κ 已高、不需複合。

因此整條研究鏈的結論收斂為：**架構（場/回熱/分層/高頻）已優化到位、複合策略
已就緒，下一個、也是唯一的量級槓桿是「把稀土（Gd/La/Mn-Fe-P）元素與其一階
相變物性納入 surrogate 與元素空間」**。一旦元素空間擴張，本 PR 的複合整機目標
會自動對「高 ΔM 稀土相 + 最佳 κ 基底」收斂——框架已備妥。

## 模型限制

- 複合 κ 採並聯/串聯 connectivity=0.7 加權，需微結構校準。
- 上述絕對值來自快速代理（Br 偏高），僅供相對趨勢；正式結論需生產級代理。
- φ 掃描為 31 點網格，足夠定位最佳分率量級。
