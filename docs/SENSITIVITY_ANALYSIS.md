# 假設參數敏感度分析（D6）

> D6 的核心是「複合/整機模型有幾個目前靠假設、無實測的參數」。與其空等實測，
> 先**量化每個假設到底影響結論多少**——藉此判斷哪些值得優先量測，哪些現值就夠用。
> 由 `scripts/sensitivity_analysis.py` 產出（材料 (Mn,Fe)₂(P,Si)、基底 Cu）。

---

## 結果

| 假設參數 | 掃描範圍 | 功率增益範圍 | 相對擺幅 | φ* 範圍 |
|---|---|---|---|---|
| **connectivity** | 0.40–1.00 | ×7.6 – ×11.8 | **43%** | 0.31–0.38 |
| extra_regeneration | 0.00–0.95 | ×10.0 – ×10.0 | **0%** | 0.34 |

## 解讀

1. **connectivity 是唯一值得優先實測的假設**（相對擺幅 43%）。它改變複合增益的
   絕對值（×7.6↔×11.8），但**不改變定性結論**——複合在整個 connectivity 範圍內
   都顯著有益（增益 > ×7），最佳基底分率 φ* 也只在 0.31–0.38 小幅移動。
   → 量測時以 SEM 量真實相連通度回填（見 `MEASUREMENT_PROTOCOL.md` 第 3 節）。

2. **回熱（extra_regeneration）在「複合增益比」中完全相消**（0% 擺幅）：回熱同時
   放大 bare 與 composite 的功率，比值不變。故回熱**不影響「要不要複合、加多少」
   的決策**；它只影響**絕對**功率密度（那本就被 D12 標記為理想化上界）。
   → 對複合決策而言，回熱假設不需優先量測。

3. **整體啟示**：複合材料這條出路的**定性結論對假設參數穩健**（增益 ×7–12、
   φ*≈0.3），不因 connectivity/回熱的具體取值而翻盤。需要實測校準的只有
   connectivity 的**絕對**增益值，且影響有界（±約 20%）。

## 復現

```bash
python scripts/sensitivity_analysis.py
```

> 與 D12 一致的訊息：本引擎的**相對**結論（複合有益、φ*、瓶頸排序）穩健可信；
> **絕對**數值（增益倍率、功率密度）才需實測收斂。優先量測對象＝connectivity。
