# Hc / σy 可預測性評估：為何停在合成（內在 vs 外在性質）

> 評估「能否像 Tc / Br 那樣，用真實資料把 Hc（矯頑力）與 σy（降伏強度）也變成
> 可預測」。結論：**不能——但這是物理本質的限制，不是資料蒐集的缺口。** 上網調研後
> 的誠實判斷如下。

---

## 1. 為何 Tc / Br 成功、Hc / σy 不行：內在 vs 外在

| 性質 | 類型 | 主要決定因素 | 公開組成級資料 | 結果 |
|---|---|---|---|---|
| Tc | **內在** | 成分/電子結構 | NEMAD（15,577，乾淨 CSV）| ✅ R²=0.78 |
| 磁化/Br | **內在** | 成分/磁矩 | Materials Project DFT | ✅ R²=0.58 |
| **Hc** | **外在** | 晶粒、缺陷、應力、相界（製程主導）| 無乾淨組成級 CSV | ❌ 組成天花板低 |
| **σy** | **外在** | 晶粒尺寸、冷加工、退火、析出 | 無乾淨組成級 CSV | ❌ 組成天花板低 |

內在性質由成分/電子結構決定，故「成分 → 性質」可學；外在性質由**微結構與製程**主導，
同一成分經不同製程可差一個數量級，**單靠成分本質上學不準**。

## 2. 文獻佐證（上網調研）

- **Hc**：確有 Fe 基軟磁資料集（885 筆 Fe-Co-Ni-Si-B-C-P-Cu…含 Hc）、Fe-Co-Ni 異質
  資料庫（鑄造/粉末冶金/電鍍/積層）。但**全都把製程參數當特徵**，因矯頑力是外在量；
  且資料含 B/C 等本元素空間外元素、嵌在論文中（非如 NEMAD 的乾淨公開 CSV）。
- **σy**：HEA 屈服強度 ML 模型 R² 可達 ~0.85，但**明確需要製程特徵**（冷軋率、退火
  溫度/時間）；資料量小（154–284 筆）、多為高熵合金族、組成級乾淨資料同樣難取。

→ 即使取得這些資料，**組成-only 模型的上限仍低**（缺製程這個主導變數），且資料不像
Tc/Br 那樣可乾淨自動抓取。投入產出比遠低於 Tc/Br。

## 3. 結論與處置

- **Hc、σy 維持合成**，作為 GA 的**軟約束/相對排序**用途（篩掉硬磁、低強度配方），
  **不宣稱為可預測的絕對值**。能力聲明已如此標示。
- 這是**物理限制的誠實記錄**：不是「還沒做」，而是「組成-only 對外在性質本質受限」。
- 若未來要做：必須**納入製程參數**為輸入（成分 + 晶粒/退火/加工），那是另一類「製程感知」
  模型，超出本引擎「成分 → 性質」的範疇。

## 4. 對整體可信度的意義

引擎的**兩個內在磁性量（Tc、Br）已真實資料化、可預測、帶誤差條**；兩個外在量
（Hc、σy）誠實地維持合成、僅作相對約束。這個分界本身就是對「什麼能預測、什麼不能」
的清楚交代——比硬湊一個低可信度的 Hc/σy「真實」模型更誠實。
