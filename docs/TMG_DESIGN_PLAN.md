# 熱磁發電機設計方案與模擬評估

> 本文整合三份基礎：正向發電 (`generator_design.py`)、反向製冷對偶
> (`magnetocaloric_refrigeration.py`)、模擬腳本 (`scripts/simulate_tmg_design.py`)。
> 回答：(1) 怎麼設計、(2) 發電原理評估、(3) 模擬結果。

## 一、設計規劃：四層架構藍圖

設計選擇全部由「反向 MCR 教訓」反推而來（見 MAGNETOCALORIC_DUALITY.md）：

```
            熱端廢熱流 (T_hot)
                 │  強制微通道熱交換器（高頻交替沖刷）
   ┌─────────────▼─────────────┐
   │  Halbach NdFeB 永磁陣列     │  ← 教訓四：B_app = 1.24–1.5 T
   │ ┌─────────────────────────┐ │
   │ │ 分層 Tc 梯度發電床 (N≈8) │ │  ← 教訓三：每層 Tc 遞增，窄子溫窗
   │ │  層1 (Tc低) ── 線圈 N匝   │ │     各層在最佳相變點工作
   │ │  層2        ── 線圈 N匝   │ │     線圈串聯 → 電壓相加
   │ │   ⋮  低矯頑力低磁滯材料    │ │  ← 教訓一：窄迴線 (Steinmetz 損低)
   │ │  層8 (Tc高) ── 線圈 N匝   │ │
   │ └─────────────────────────┘ │
   │  固態導熱回熱器 (高導熱 Cu/Al) │  ← 教訓二：全固態，免泵浦寄生損
   └─────────────▲─────────────┘
                 │
            冷端冷卻水 (T_cold)
```

| 層 | 設計決策 | 數值/依據 |
|---|---|---|
| 永磁 | Halbach 閉環陣列 | B_app=1.4 T（MCR 室溫機常用 1.24–1.5T） |
| 材料 | 分層 Tc 梯度，低 Hc/低磁滯 | N≈8 層，每層子溫差 ~7.5K |
| 回熱 | 全固態導熱回熱器 | ε≈0.90，避開水基 50% 泵浦損 |
| 機構 | 薄板高頻 | L=0.5 mm → f≈15 Hz（f∝1/L²） |
| 取電 | 各層線圈串聯 | V = ΣV_i（電壓相加） |

## 二、發電原理評估

發電鏈：**廢熱 → 材料反覆跨越 Tc → delta_M 磁化變化 → 切斷/接通 Halbach
磁通 → 線圈感應 V=-N dΦ/dt → 電。** 物理上成立，但有兩個結構性瓶頸：

1. **效率被顯熱主導**（正向死穴）：q_in 幾乎全花在「把整塊金屬反覆加熱」，
   磁功只佔萬分之幾。對策＝降 Cp、窄子溫窗（分層）、固態回熱。
2. **磁滯吃掉淨功**（與反向共通死穴）：Steinmetz P_hyst∝f·Hc·Br，高頻下更嚴重。
   對策＝選低矯頑力窄迴線材料。

**結論：原理可行，但這是「低效率、拚功率密度」的技術**，最適合 100°C 以下、
熱源免費的廢熱回收，不適合追求高轉換效率的場景。

## 三、模擬結果（`scripts/simulate_tmg_design.py`，三廢熱情境）

| 情境 | 設計 | η (%) | η/η_C (%) | 功率密度 (kW/m³) | V_rms (V) |
|---|---|---|---|---|---|
| 低溫 150°C | 保守 (1T,無回熱,1mm) | 0.022 | 0.2 | 735 | 0.14 |
| | 升級單層 (1.4T,ε0.9,0.5mm) | 0.292 | 2.2 | 4,114 | 0.55 |
| | **分層+回熱 N=8** | **1.560** | **11.8** | **4,114** | **4.37** |
| 中溫 350°C | 分層+回熱 N=8 | 1.284 | 14.0 | 4,133 | 4.39 |
| 高溫 500°C | 分層+回熱 N=8 | 1.125 | 15.1 | 3,942 | 4.19 |

**讀數**：
- 架構優化（場↑、回熱、薄板、分層）把效率從 0.022% 拉到 1.56%，**約 71 倍**；
  功率密度 ×5.6（0.73→4.1 MW/m³）；電壓 ×31（串聯 8 層，0.14→4.37 V）。
- 但 η 絕對值仍只有 ~1–1.6%。**主因是 delta_M=0.20 T 的 Fe 系材料是硬天花板**
  ——這跟 README 已知限制一致（工業 Gd 可達 0.6T，一階 La-Fe-Si/Mn-Fe-P 更高）。
- 兩條提效路線（單層大回熱 vs 分層）效率相近，但**分層額外白賺 8× 電壓**，
  且不需要單一超高效回熱器，工程上更穩健。

設計旋鈕敏感度見 `results/tmg_design/sweeps.png`，可逆性（同材料正向發電
vs 反向製冷）見 `reversibility.png`。

## 四、下一步可一起思考的方向

1. **換材料**：把 delta_M 從 0.2T 提到 0.5–1T（稀土/一階相變），是效率量級
   提升的唯一硬解——可在 `properties.py` 加入 Gd/La-Fe-Si 校準後重跑。
2. **共生機**：同一套硬體白天發電、夜間（或谷電）反向製冷，用對偶模組同時設計。
3. **磁路成本**：引入類 OptiMag 的磁-熱耦合優化，削減 NdFeB 用量。
4. **GA 目標函數**：把本模組的整機功率密度/效率接回 GA fitness，讓搜尋直接
   對「整機」而非「單一材料」最佳化。
